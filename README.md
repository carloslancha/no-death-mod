# No Death Mod

> 段落引用

# Requirements
- Hollow Knight 1.4.3.2
- Modding API 1.4.3.2-60

## How to install

Unzip the [last release](https://github.com/carloslancha/no-death-mod/releases/latest) .zip file in `HollowKnightPath/hollow_knight_Data/Managed/Mods` folder.
